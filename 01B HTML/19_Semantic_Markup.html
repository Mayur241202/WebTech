Semantic Markup

It is the markup that relates to the meaning of content. There are two types :

  1. Semantic
  2. Non-semantic

Semantic Tags

Semantic tags are the tags from which we are able to classify which type of content it has or it includes.

Example : heading tag, paragraph tag, image tag, etc.

Non-semantic Tags

Non-semantic tags are the tags from which we are not able to classify or identify which type of content it has or it includes.

Example : div tag, span tag, etc.

Note :

Due to semantic tags our website is more meaningful and structured layout. It also helps browser in understanding our website i.e. browser can arrange or ranked websites based on information i.e. content given in semantic tags of a website. That means our website is SEO friendly. And due to this our website is more readable. Due to this user experience is increase.
