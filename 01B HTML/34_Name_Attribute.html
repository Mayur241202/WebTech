Name Attribute In HTML Forms

Name of the form control. Submitted with the form as a part of a name/value pair.

After entering username and password, if you see in url after submitting form then you will get http://127.0.0.1:3000/action?user=abc&pass=123 where username and password is written. Here name is user or pass and value is abc or 123 repectively. That is they are in name/value form.
