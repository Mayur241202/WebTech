Div Element

Div is a container used to hold other HTML elements or group of elements together. It is a Block element. It is also called as content division element.

Without Div : Anchor Tags

Orange Apple Banana

With Div : Anchor Tags

Orange Apple

Orange and Apple has same div block.

Banana

Banana has seperate div block.

Note : We are able to do this for all types of block and inline elements.
