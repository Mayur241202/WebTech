Span Element

Span is also a container used to hold other HTML elements or group of elements together. But it is an Inline element.

Without Span : Anchor Tags

Orange Apple Banana

Here all anchor tags has not in any container i.e. they are seperate to each other.

With Span : Anchor Tags

Orange Apple Banana

Orange and Apple are in same Span block i.e. they are in a same container, but due to span is an inline element, banana is appear on the same line but banana has not any div or span block although we are able to do that and that not affect the result.

Note : For better understanding use inspect feature.
