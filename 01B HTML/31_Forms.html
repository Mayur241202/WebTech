Forms In HTML

We use form tag to create a form. Forms are used to collect data from the user.

Action Attribute

action attribute is used to define what action needs to be performed when the form is submitted or where the form data should be sent.

Input : Form Element

  • Used to create multiple form controls.
  • There are multiple types of inputs that can be created using type attribute.

type attribute values

  • By default : text
  • text
  • password
  • number
  • time
  • color
