Tables In HTML

Tables are used to represent real life table data. There are four main entities while creating a table. Before use this entities we have to use table tag to create a table.

  • caption(title) : created by caption tag
  • rows : created by tr tag used to display table rows
  • data : created by td tag used to display table data
  • header : created by th used to display header of table header

Example :

Food Menu
Item Price
Fruity 20
Samosa 12
Chips 10

Note : We mostly use tr tag outside th and td tags to create table rowise.

Note : For making borders for table we have use CSS.
