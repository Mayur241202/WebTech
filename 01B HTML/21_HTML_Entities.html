HTML Entities

  • An HTML entity is an piece ot text or string that begins with an ampersand(&) and ends with a semicolon(;).
  • Used to display reserved characters(which would otherwise be interpreted as HTML code), and invisible characters(like non-breaking spaces).
  • Can also use in place of charactes that are difficult to type with a standard keyboard.
  • Browser interprets them and renders correct charecter.

Less Than

Less than is created by lt.

Example

a < b

Note : Similary for greater than we use gt, for ampersand we use amp and for quotes we use quot on both side ot text.

Note : Similarly for characters or symbols that are not directly typed through keyboard like heart, star, sun, etc have some codes or names to display them on screen.
