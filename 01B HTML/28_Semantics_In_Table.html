Semantics In Tables

TThere are three semantics used in table.

  • header : created by thead tag used to wrap header part of table
  • body : created by tbody tag used to wrap all the rows containing table data
  • footer : created by tfoot tag used to wrap footer of the table like sum, average, etc.
