Comments

Comments are ignored by editor as they are not contribute in code. We are able to write anything in comments which we want. Usually used for writing something about code so that we can able to understand it.

HTML : Not Case Sensitive

HTML is not a case sensitive language i.e. we are able to write HTML code in small or capital words like we are able to write p tag as P and heading h1 as H1 and so on.
