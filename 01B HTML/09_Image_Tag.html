Image Tag

Used to add image to our page. It is not a pair tag like heading tag, paragraph tag, etc.

Here we have not to add any input text like we did in anchor tag or any other pair tag.

Image tag consist of some attributes like src which is used to include relative URL(We have to save image in same folder in which our html file saved.) or absolute URL(We have to add url of the image which avilable on internet but if that image was deleted from internet then our image is also not shown in our website.)

Second main attribute of image tag is alt(alternate title) which is used for alternate name if image is not shown on the page due to some issue.

Also we have some attributes to adjust the image size to fit in page like height and width.
