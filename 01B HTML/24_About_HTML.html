Understanding HTML5

The term HTML is essentially a buzzword that refers to a set of modern web technologies. This imcludes the HTML Living Standard, along with Javascript APIs to enhance storage, multimedia and hardware access. This includes new features or upgrades in HTML and other features or things not part of HTML.

How HTML Works?

HTML standard is a document that tells to the browser how HTML should work. This is always available with HTML. Due to this browser can render webpages.
