HTML means Hyper Text Markup Language.

HTML Boilerplate is the standard format or skeleton to write HTML Code.

html tag is the root tag and has attribute lang="en" to show en means english language is used to write code.

head tag includes all metadata i.e. data about data and metadata has some attributes like charset, name, etc with their values in single quote or double quote. Also includes title tag which represents title of website.

html tag has also body tag which includes all tags like elements like paragraph, headings, etc.
