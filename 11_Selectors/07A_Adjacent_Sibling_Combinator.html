Adjacent Sibling Combinator

This selects an elements which are coming just after the current tag/element. For example, Heading 3 that comes immediately after paragraph. We access that by using (+) symbol in between the elements. We are also able to use class and id in place of elements.

Syntax :

p + h3 {
            property : value;
}
