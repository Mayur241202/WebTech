Practice - Specificity

Q. 1 : For the given HTML and CSS code what is color of h1 ?

ans. : For this we have to calculate specificity.

h1 has specificity (0, 0, 1) as h1 is an element. #greet has specificity (1, 0, 0) as greet is an Id. .myClass has specificity (0, 1, 0) as myClass is a class. Therefore h1 has yellow color.

Q. 2 : For the given HTML and CSS code what is background-color of p ?

ans. :

div p has specificity (0, 0, 2) as div and p are 2 elements. .para has specificity (0, 1, 0) as para is a class. p has specificity (0, 0, 1) as p is an element. Therefore p has background-color blue.

Q. 3 : For the given HTML and CSS code what is color of text "Hello world!" ?

ans. :

div.greetDiv .greet has specificity (0, 2, 1) as div is an element, greetDiv and greet are two classes. .greetDiv p.greet has specificity (0, 2, 1) as p is an element, greetDiv and greet are two classes. .greetDiv .greet has specificity (0, 2, 0) as greetDiv and greet are two classes.

Here specificity of div.greetDiv .greet and .greetDiv p.greet are same and hence here we apply cascading. Therefore final answer is depend on .greetDiv p.greet as this will come after div.greetDiv .greet. Therefore "Hello World!" is in blue color.
