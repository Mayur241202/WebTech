Pseudo Elements

A keyword added too a selector that lets you style a specific part of a selected elements like first letter of the heading, first line of the paragraph or when we select a text from paragraph its background color, etc.

It is implement by using double colon after element.

Syntax :

p::first-letter {
        property : value;
}

Example

Heading 2 First Letter

First Line : Lorem ipsum dolor sit amet consectetur adipisicing elit. Cumque deserunt, cum quisquam accusamus corporis pariatur nemo, aliquid laborum facere voluptatum voluptas enim, distinctio consequuntur eum voluptates odio necessitatibus est facilis.

For selection select a text from any paragraph you will see a yellow color.
