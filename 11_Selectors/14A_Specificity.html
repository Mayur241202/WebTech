Specificity

Specificity is an algorithm which calculates thwe weight that is applies to a given CSS declaration. It is calculate by number of id's at hundreds place, class + pseudo class + attributes at tens place, elements + pseudo elements at one's place specify in selector while writing code of styling.

(No. of Id's, No. of class + pseudo class + attribute, No. of elements + pseudo elements)

Example

.myClass:hover {
           property : value;
}

Here .myClass:hover include one class and one pseudo class, therefore its specificity is (0, 2, 0).

h2 {
     property : value;
}

Here h2 is an element and therefore its specificity is (0, 0, 1).

If specificities are same for more than one codes of same elements then cascading is applied i.e. whichever is declared later will have highest priority.

Note : If we include inline style to element then inline style has highest priority than any other declarations.

Note : inline > id > class > element
