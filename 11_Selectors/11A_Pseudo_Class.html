Pseudo Class

A keyword added to a selector that specifies a special state of the selected elements. For example, if we want to change the color of text or background if we hover on that element. There are multiple keywords that are added to a selector hover, active, checked, nth-of-type, etc.

Syntax :

button:hover {
             property : value;
}

Example (hover)

hover : Only when we take a cursor on element it apply an style on that element. For following example if you go on submit button then its background color change to green and text color to white.

Example (active)

active : If we click on button then only it can apply an style. Similar for other elements also. For following example if we click on paragraph then its color changes to pink. Actually this is apply for all the elements which are in paragraph tags.

Mostly used for radio buttons. If we check a radio button then only style will apply.

Example (nth of element)

nth of element is used for access elements in a particular pattern like odd elements, even elements, multiple of 3 elements, etc.
