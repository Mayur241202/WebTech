Inheritance

Inheritance in CSS means that some properties of an ancestor element (parent, grandparent, etc.) automatically apply to its child elements unless overridden.

Example

If you set a text color (color: red;) for a div, all text inside that div (like in p, span, etc.) will also be red unless you override it.

The p inside the div will have red text without explicitly setting color: red; inside p.

  • Inherited Properties:
    • color
    • font-family
    • visibility
    • cursor

  • Not Inherited (But Can Be):
    • border
    • margin
    • padding
    • background
    • width, height

Forcing Inheritance

Use inherit to manually inherit a property :

Example

            
            p {
                color: inherit; /* Takes the color of the parent */
            }
