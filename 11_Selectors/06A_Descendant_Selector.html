Descendant Selector

We are able to select a particular element or elements even if they are in another element or elements. For example, if paragraph is written in a div tag then also we are able to access that paragraph by using descendant selector.

Syntax :

div p ol li {
          Property : value;
}

Here we are accessing list item tags to apply CSS on it.
