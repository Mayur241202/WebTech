Element Selector

Select all elements of same type.

h1 {
       property : value;
}

In place of h1 we are able to write any element which is declared in out html document also if we have to give same css code to more than one elements then we have to write all the elements in place of h1 seperating with comma.

Example

Heading 3

Heading 4

In Heading 2 we change the color and for Heading 3 and 4 we use same code as both want color aqua. For better understanding see the CSS code.
