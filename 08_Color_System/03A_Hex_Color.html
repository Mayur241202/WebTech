Hex Color

Hex means hexadecimal i.e. here we use hexadecimal number system to create a color.

Example

#FFFFFF is a white color. Here 1st 2 F's denote red color, next 2 F's denote green color and next 2 F's denote blue color similar to RGB color codes. We can able to write hexadecimal characters in place of F to change the color.

Hi there!(#FF0000)

#abcdef

#458bfa

Hexcode range is from 00 to FF and it is not case sensitive. If there are two same characters for red color then we denote it as 1 same character.

Example

#ff00ff : #f0f

#ffffff : #fff
