Font Size

There are two types of font size values. First is absolute i.e. they are not changes in any condition and relative i.e they are declare relative to other properties.

Absolute values has units like px, pt, pc, cm, mm, in, etc. from which we are mostlyh use px.

Relative values has units like %, em, rem, ch, vh, etc.

1 in = 96 px

1 in = 72 pt

12 pt = 1 pc

Pixels(px)

Most commonly used absolute unit. It is not suitable for responsive websites.
