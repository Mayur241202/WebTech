Inline v/s Block Elements

Display Property

Used to make a block Elements an inline Elements and vice versa. Also it has third value block-inline.

Syntax

        
            h2 {
                display : inline;
            }

Example

Heading 1

Heading 2

Span 1 Span 2

Here heading is changes to inline element.

Note : Third value block-inline is used when we use span element inside a div element and give them some properties like height, width, margin, padding, etc. From this properties some are not applied and thats why we use block-inline value for display property.
