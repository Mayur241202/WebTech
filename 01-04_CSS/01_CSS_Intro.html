CSS :

Full Form : Cascading Style Sheet

If HTML is a human skeleton then human skeleton with skin and cloths is CSS.

It is a language that is used to describe the style of a document.

It has some properties like text, color, background color, etc. to style the HTML Page.

Basic Format Of CSS

h1(Selector){
        color(Property) : red(value);
}

Here Selector is an element which is defined in HTML document and we are applying color property to color the heading h1 by red color.
