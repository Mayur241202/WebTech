Why Not Internal CSS?

If our website is big enough so that we want number of HTML pages then to style them again we have to write internal CSS for each HTML page. To avoid this we mostly prefer External CSS.

External CSS

Writing CSS in a seperate document and link it with HTML file.

After creating seperate CSS file we have to link it with HTML file by using link tag which is defined in head tag. This will demostrate in next html file.

link tag includes two main attributes rel(relation) and href(reference link) of CSS file.
