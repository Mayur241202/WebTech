Why Not Inline CSS

We do not use inline CSS as if we want same property apply to another element then we have again write same code for that element.

In above all the heading are h1 headings and we use style property to color them with red color. To apply this style we have to write style="color: red;" this code 3 times as headings are three which is time consuming. Therefore we do not use inline CSS most of the times.

Internal CSS

Here style is added using the style element declared in the same document and mostly in head tag. Due to this we are able to style all elements with same code used for multiple times.
