Transform Properties

This property lets you Rotate, Scale, Skew or translate an element.

Rotate

It rotates an element by some degree or radian, etc.

Syntax

        
            transform : rotate(45deg);

Example

ONE

Here we rotate div by 180 degree. We are able to rotate it with negative value also.

Scale

It scales i.e. increase or decrease a size of an element in X and Y direction by some sie relative to original size of an element.

Syntax

        
            transform : scale(2, 0.5);

Example

TWO

Here we reduce size of div by scale(0.5). It reduce size in both directions i.e. X and Y.

TWO.v1

Here we change size of div in X and Y direction differently by using scale(2, 0.5). It half size in Y direction and double size in X direction. We also able to use scaleX() and scaleY() to change sizes in X and Y direction respectively.

Translate

It moves an element in X and Y direction.

Syntax

        
            transform : translate(100px, 50px);

Example

THREE

Here we translate div i.e. move div by 50 px in X and Y direction. Similar to scale we are able to use translateX() and translateY().

Skew

It will tilt an element by some degrees in X or Y direction. We are able to use negative values also.

Syntax

        
            transform : skew(30deg);

Example

FOUR

Here we tilt div by 30 degrees.

Example : Rotate & Translate

FIVE

Here we rotate div by 45 degrees and then use translation in X direction by 50px.
