Alpha

Sets the opacity i.e. visbility for a color. Its values ranges from 0 to 1. Here 0 means hidden and 1 means visible. It is used with rgb value of a color with additional a is added at the end i.e. rgba(red, green, blue, alpha). We also able to use hex values like #ff000066. Here last 66 represents alpha value or opacity of a color.

Example

Heading

rgba(255, 0, 0, 0.40);

Here heading color did not affect by div color.

Opacity

Sets the opacity of the element ranges from 0 to 1. Same as alpha values.
