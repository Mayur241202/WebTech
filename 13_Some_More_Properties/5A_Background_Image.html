Background Image

Sets the background-image for an element. Also to set the image in the element we use background-size and it has three values contain, cover and auto which is default value also.

Syntax

        
            div {
                background-image: url("5A_img_1.jpg");
                background-size: contain or cover or auto;
            }

Example 1

Auto

Here we use and background-size: auto; Auto will stretch an image.

Example 2

Contain

Here we use and background-size: contain; contain will repeat the same image to fill the element.

Example 3

Cover

Here we use and background-size: cover; cover will crop the image to fill the element.
