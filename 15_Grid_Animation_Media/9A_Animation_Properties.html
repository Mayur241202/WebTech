Animation Properties

Firstly we have to create an template which is given above and then we have declare following propeties for the element for which we want to give animation.

Example

This is a Box.
        
            .box {
                height: 100px;
                width: 300px;
                background-color: pink;
            
                /* Animation Properties For fontAnimation */
            
                animation-name: fontAnimation;
                animation-duration: 3s;
                animation-timing-function: ease-in;
                animation-delay: 1s;
                animation-iteration-count: 5;
                animation-direction: alternate;
            }
            
            /* Animation Template */
            @keyframes fontAnimation {
                from {
                    font-size: 20px;
                }
            
                to {
                    font-size: 40px;
                }
            }

Try to change the animation template and some propety values for better understanding.

Animation Shorthand

Example

This is a Box.
        
            .box1 {
                height: 100px;
                width: 300px;
                background-color: pink;
            
                /* Animation Properties For fontAnimation */
            
                animation: fontandColorAnimation 5s ease-in 1s 5 alternate;
            }
            
            /* Animation Template */
            @keyframes fontandColorAnimation {
                from {
                    font-size: 20px;
                    background-color: aqua;
                }
            
                to {
                    font-size: 40px;
                    background-color: yellowgreen;
                }
            }

Try to change the animation template and some propety values for better understanding.

% In Animation

Example

This is a Box.
        
            .box2 {
                height: 100px;
                width: 300px;
                background-color: pink;
            
                /* Animation Properties For fontAnimation */
            
                animation: perfontandColorAnimation 5s ease-in 1s 5 alternate;
            }
            
            /* Animation Template */
            @keyframes perfontandColorAnimation {
                20% {
                    font-size: 10px;
                    background-color: aqua;
                }
            
                60% {
                    font-size: 50px;
                    background-color: yellowgreen;
                }
            
                100% {
                    font-size: 30px;
                    background-color: red;
                }
            }

Try to change the animation template and some propety values for better understanding.
