Media Queries

Help to create a responsive website.

Example

Media Queries Demo

THis is apply when width is exact 400px

Syntax

        
            /* At exact width 400px*/
            @media (width: 400px) {
                #head1 {
                    background-color: red;
                }
            }

Example

Media Queries Demo

This is individual and not a range i.e. when width is less than 400px then color is red and if width is greater than 800px then also color is red otherwise color is yellow.

Syntax

        
            /* At greater than width 800px */
            @media (min-width: 800px) {
                #head2 {
                    background-color: red;
                }
            }

            /* At less than width 400px */
            @media (max-width: 400px) {
                #head2 {
                    background-color: greenyellow;
                }
            }

Example

Media Queries Demo

This is a range i.e. when width is in the range of 400px to 800px then color is red otherwise color is yellow.

Syntax

        
            /* Red color between Range 400px to 800px */
            @media (min-width: 400px) and (max-width: 800px) {
                #head3 {
                    background-color: red;
                }
            }

Media Queries Demo

Orientation means rotating the phone and it has two types landscape and portrait. It changes color to red when phone is landscape. and in portrait it changes to yellow.

Syntax

        
            /* Red color between Range 400px to 800px */
